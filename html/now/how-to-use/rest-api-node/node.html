Node¶

Подсказка

Правила формирования запросов к ноде приведены в разделе REST API ноды.

GET /node/config¶

Возвращает основные конфигурационные параметры ноды.

Ответ метода:

{
    "version": "0.6.6",
    "waves-crypto": false,
    "chainId": "D",
    "consensus": "POS",
    "minimumFee": {
        "1": 0,
        "3": 100000000,
        "4": 100000,
        "5": 100000000,
        "6": 100000,
        "7": 300000,
        "8": 100000,
        "9": 100000,
        "10": 100000,
        "11": 100000,
        "12": 100000,
        "13": 1000000,
        "14": 100000000,
        "15": 100000000,
        "102": 0
    }
}

POST /node/stop¶

Запрос останавливает ноду.

GET /node/status¶

Возвращает основные конфигурационные параметры ноды.

Ответ метода:

{
    "blockchainHeight": 47041,
    "stateHeight": 47041,
    "updatedTimestamp": 1544709501138,
    "updatedDate": "2018-12-13T13:58:21.138Z"
}

GET /node/version¶

Возвращает версию приложения.

Ответ метода:

{
  "version": "Waves Enterprise v0.9.0"
}
